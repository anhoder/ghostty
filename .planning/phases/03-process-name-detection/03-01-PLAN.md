  <action>
Create new file `src/os/process.zig` with platform-specific foreground process name detection.

**Public API:**
```zig
pub fn getForegroundProcessName(
    alloc: Allocator,
    pty_master_fd: posix.fd_t,
) !?[]const u8
```
Returns allocated process name string or null if no foreground process. Caller owns returned memory.

**Linux implementation (getForegroundProcessNameLinux):**
1. Call `std.c.tcgetpgrp(pty_master_fd)` to get foreground PGID
2. Return null if pgid <= 0
3. Iterate `/proc` using `std.fs.openDirAbsolute("/proc", .{ .iterate = true })`
4. For each numeric directory, parse PID and check if `std.c.getpgid(pid) == pgid`
5. When match found, read `/proc/<pid>/comm` (max 16 bytes, trim trailing newline)
6. Return allocated duplicate of process name
7. Handle errors gracefully (continue on permission denied, return null if no match)

**macOS implementation (getForegroundProcessNameBSD):**
1. Call `std.c.tcgetpgrp(pty_master_fd)` to get foreground PGID
2. Return null if pgid <= 0
3. Use `@cImport({ @cInclude("libproc.h"); })` for libproc types
4. Call `proc_listallpids()` to get all PIDs (allocate buffer for 4096 PIDs)
5. For each PID, check if `std.c.getpgid(pid) == pgid`
6. When match found, call `proc_pidinfo(pid, PROC_PIDTBSDINFO, 0, &bsdinfo, sizeof)`
7. Extract `bsdinfo.pbi_name` (null-terminated, max 16 chars) using `std.mem.sliceTo(&bsdinfo.pbi_name, 0)`
8. Return allocated duplicate of process name
9. Handle errors gracefully (continue on permission denied, return null if no match)

**Platform dispatch:**
Use `switch (builtin.os.tag)` to route to Linux vs macOS implementation. Return null for unsupported platforms (Windows).

**Imports needed:**
- `std.mem.Allocator`
- `std.posix`
- `builtin`
- Platform-specific: `std.fs` (Linux), `@cImport` for libproc (macOS)

**Error handling:**
- Gracefully handle permission denied (continue iteration)
- Gracefully handle process exit races (continue iteration)
- Return null rather than error for "no process found" case

Follow Ghostty patterns: scoped logger, self-referential struct pattern not needed (pure functions), explicit allocator passing.
